采购域 AI Agent 架构

Supervisor 路由分发 → 域 Agent 多 Skill 协同执行 → 结果聚合回传
Layer 1 · Supervisor

Supervisor Agent

采购域统一入口，意图识别 → 任务分流 → 上下文传递 → 结果汇总

意图识别

理解用户自然语言输入，识别所属采购域（供应商 / 寻源 / 变更 / 价格 / 合同等），支持模糊表达消歧

任务分流

根据意图路由到对应域 Agent，跨域请求拆解为子任务编排，支持并行 / 串行调度

上下文管理

维护多轮对话状态，携带供应商名称 / RFQ 编号 / 物料号等上下文，避免重复询问

结果聚合

多 Agent 返回结果统一格式化输出，冲突结果仲裁，异常兜底 & 人工升级
