模块概述 Module Overview

采购执行管理平台（Procurement Execution Platform）是 E2E-SCM 供应链管理系统的核心模块之一，覆盖从采购申请（PR）到付款结算（Payment）的全流程。本模块集成了三单匹配引擎（Three-Way Matching Engine）、审批流模拟器（Approval Flow Simulator）和付款计算器（Payment Calculator），支持采购业务的数字化执行与异常管理。

功能模块 Functional Modules

📈

采购工作台 Dashboard

汇总展示待处理 PR、待确认 PO、待收货、待付款等关键指标，提供全流程状态跟踪和最近动态。

📝

PR管理 Purchase Request

管理采购申请单的创建、审批、转PO流程。支持多级审批、金额阈值路由。

💳

PO管理 Purchase Order

采购订单全生命周期管理，包括创建、确认、发货跟踪、收货确认和归档。

🚚

收货管理 Receiving

ASN到货通知、GRN收货单创建、IQC来料检验，支持批量收货与质量追溯。

💰

对账结算 Settlement

三单匹配（PO/GRN/Invoice）、付款计划、发票校验。含付款计算器支持早付折扣分析。

引擎 & 工具 Engines & Tools

⚙

匹配引擎 Matching Engine

三单匹配（Three-Way Matching）核心引擎。输入 PO/GRN/Invoice 金额，自动执行数量容差（±5%）、金额容差（±3%）、价格容差（±2%）检查，输出匹配结果并生成异常报告。

🔄

审批流 Approval Flow

可配置的审批规则引擎：按金额阈值自动路由至对应审批人。支持四级审批链：采购员自审 (<5万) / 采购经理 (5-50万) / VP+总监 (50-200万) / CEO+CFO (>200万)。含审批模拟与升级逻辑。

⚠️

异常处理 Exception Handling

自动检测延迟交货、质量异常、价格偏差、数量差异等异常，生成异常单并跟踪处理时间线。

💰

付款计算器 Payment Calculator

输入发票金额与付款条款，自动计算早付折扣金额、折扣日期、有效年化利率（Effective APR），并以时间线可视化对比早付 vs 按期付款。

📊 审批升级逻辑 Escalation Rules

• 审批超时 24 小时未处理 → 自动发送催办提醒 (Reminder)

• 审批超时 48 小时未处理 → 升级至上级审批人 (Escalation)

• 审批被驳回 → 退回申请人修改，重新提交进入审批流

• 四级审批（>200万）需 CEO 和 CFO 双签，任一方驳回则流程终止

• 紧急采购可标记 紧急 跳级审批，但需事后补签
