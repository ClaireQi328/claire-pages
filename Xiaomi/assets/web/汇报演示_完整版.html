🏗️ AI Agent · 四层混合架构

定位：以"采购数字员工"形成主动智能（非 Chatbot）——人做决策 = min(人工操作) × max(决策质量) × min(流程路径)

L1Coordinator Agent（总协调层） — 全局任务编排、需求智能分流、异常升级路由、跨Agent协同调度
L2Domain Agent（领域子Agent层） — 需求解析Agent / 订单执行Agent / 订单追踪Agent / 质量评估Agent / 成本分析Agent
L3Skill 层（可复用能力单元） — 供应商匹配、批量下单、价格检索、BOM解析、异常分流通知、飞书卡片推送
L4CLI 层（原子API接口） — ep demand list / ep order create / ep supplier search / feishu card send 等 30+ 原子接口
